class InjectExtrinsicParameters extends RenderParameter

This is a static class with static intrinsic/extrinsic parameters which are going to be injected into RenderParameter object. Unless you change the resolution of the image for the Camera, those parameters will always be the same as bellow. ImageSize -> 640x480
